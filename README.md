# IsTyping...
A simple Bash script to simulate "is typing..." in Telegram(desktop)

First install xdotool
```
sudo apt-get update
sudo apt-get install xdotool
 ```
Then clone or download the repo and make sure Telgram desktop is open.

Run:
```
cd IsTyping/
./IsTyping
```
